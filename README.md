# ball-toy4

A mashup/experiment of a rolling ball with grapple and jump abilities with a tree dropping apples (multiple project combine)

Playable at https://dcrep.github.io/ball-toy4/

Controls: standard Left/Right arrow or A/D move, space-bar to jump or launch off grapple, left-click to grapple to nearest grapple point

Source at https://github.com/dcrep/mod1-w4toyrepo-DC
